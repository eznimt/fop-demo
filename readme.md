### Main ausführen

Es werden PDF-Dateien ins Verzeichnis "pdf" geschrieben, für Apache FOP und für LaTeX.

### Dateien

#### fop/01_hello_world.xml

* typisch simples "Hello World"

#### fop/02_data.xml

* Erweiterung mit einer Tabelle
* Tabelle enthält Daten (statisch)

#### fop/03_hello_world_with_params.xml

* wie "Hello World"
* Erweiterung durch einen Parameter, der im Java-Code übergeben wird

#### fop/data_from_java.xml

* wie 02_data.xml
* Tabelle ohne Body
* Daten können nicht ohne weiteres als Parameter übergeben werden

#### latex/01_document.tex

* Auszug aus einem Dokument mit Inhaltsverzeichnis und Fußnoten
